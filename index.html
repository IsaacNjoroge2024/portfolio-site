Tradie

Tradie is a production-grade autonomous trading bot built on a microservices architecture. It ingests real-time signals from custom TradingView Pine Scripts, validates them through a strategy engine with strict risk rules, filters out high-risk market conditions using NLP-powered sentiment analysis, and executes bracket orders through Interactive Brokers — all while sending live trade alerts via Telegram.

  • • Microservices: Spring Boot 3.x (Java 21) API gateway, strategy engine & order executor
  • • Python/FastAPI sentiment service with VADER & FinBERT NLP models
  • • Apache Kafka for real-time inter-service event streaming
  • • TradingView Pine Script signals using Smart Money Concepts (FVG, Order Blocks, BOS/CHoCH)
  • • Interactive Brokers TWS API with bracket order execution & position management
  • • TimescaleDB, Redis, Docker, Prometheus & Grafana for data and monitoring
